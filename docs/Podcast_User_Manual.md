# 塔罗水晶运势播客 - 用户操作手册

## 1. 功能简介
本页面为您提供每日专属的塔罗水晶运势播客。您不仅可以收听运势解读，还可以实时查看对应的字幕内容。

## 2. 界面说明
页面采用神秘学风格设计，包含以下主要区域：
1.  **标题区**: 显示当前播客的标题及发布日期。
2.  **播放器**:
    -   **水晶图标**: 播放时的动态视觉元素。
    -   **进度条**: 显示并控制播放进度。
    -   **控制按钮**: 播放/暂停，快进/快退 15秒。
    -   **倍速控制**: 支持 0.75x, 1.0x, 1.25x, 1.5x, 2.0x 播放速度。
3.  **字幕区**: 
    -   实时滚动显示当前对话内容。
    -   当前播放的句子会高亮显示。
    -   区分说话人（先生/女士/旁白）。

## 3. 操作指南

### 3.1 播放控制
- 点击中间的 **播放/暂停** 按钮开始或停止播放。
- 点击 **-15** 或 **+15** 按钮可以快速后退或前进 15 秒。
- 拖动或点击 **进度条** 可跳转到指定时间。
- 点击下方的 **倍速按钮** (如 1.5x) 可调整播放速度。

### 3.2 字幕浏览
- 字幕会跟随音频自动滚动，始终保持当前句子在视野范围内。
- 您也可以手动滑动字幕区域查看上下文。

## 4. 常见问题
- **Q: 为什么没有声音？**
    - A: 请检查手机音量是否开启，或是否处于静音模式。
- **Q: 字幕不滚动了？**
    - A: 如果您手动滑动了字幕，它可能会暂时停止自动滚动。点击播放器控件通常会恢复同步。
- **Q: 显示“获取数据失败”？**
    - A: 请检查您的网络连接。如果问题持续，可能是服务器暂时不可用。
